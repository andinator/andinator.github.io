First big Swift project

September 24, 2016 by Andi

It's kinda akward. I never felt like blogging anything about my life. But as i started coding i thougt i could share my coding experiences which may help others who also want to start coding.

Actually, im not new to coding.

According to my GitHub Profile i started coding back in 2014. Well, coding might be not the correct expression for the things i did. I started with HTML and some CSS at a very basic level.

But i got bored rather fast because i did not know that modern websites do not only use HTML and CSS. So i searched for a "real" programming language and found Java. It was at first sight. I spend about 2 years teaching java myself. I even did a project for my finals back in 2015 with Java.

I would still describe myself as a hobby Java programmer. There are still so much things to learn, espacially about GUI's. I never did much with GUI's in Java, i always found them annoying. In 2016 i started learning Swift. Swift is the programming language which Apple introduced in 2014. But it was very hard for me to begin coding because like Java i had to teach it to myself . I was really frustrated at the beginning and i were not able to do any more then a simple GUI which prints something in the console. Currently im working on a small project called "Pro-Chemistry" which is basically a Mol calculator. You can read more about my current projects here.
